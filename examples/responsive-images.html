3. Template Code for Your Projects

Copy and adapt these patterns for your actual Cloudinary URLs:

Hero Image Template

<img
  src="https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_1200,h_400,q_auto,f_auto/<PUBLIC_ID>"
  srcset="
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_800,h_267,q_auto,f_auto/<PUBLIC_ID> 800w,
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_1200,h_400,q_auto,f_auto/<PUBLIC_ID> 1200w,
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_1600,h_533,q_auto,f_auto/<PUBLIC_ID> 1600w"
  sizes="100vw"
  width="1200" height="400" 
  alt="Project hero image"
  decoding="async"
/>

Card/Thumbnail Template

<img
  src="https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_400,h_300,q_auto,f_auto/<PUBLIC_ID>"
  srcset="
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_300,h_225,q_auto,f_auto/<PUBLIC_ID> 300w,
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_400,h_300,q_auto,f_auto/<PUBLIC_ID> 400w,
    https://res.cloudinary.com/<YOUR_CLOUD>/image/upload/c_fill,w_600,h_450,q_auto,f_auto/<PUBLIC_ID> 600w"
  sizes="(max-width: 768px) 100vw, (max-width: 1200px) 50vw, 33vw"
  width="400" height="300" 
  alt="Project thumbnail"
  loading="lazy" 
  decoding="async"
/>

4. Key Optimization Features

  • q_auto - Automatic quality optimization
  • f_auto - Automatic format selection (WebP, AVIF when supported)
  • c_fill - Smart cropping to exact dimensions
  • loading="lazy" - Defers loading for below-fold images
  • decoding="async" - Non-blocking image decode
  • srcset - Multiple resolutions for different screen sizes
  • sizes - Tells browser which srcset to use based on viewport
